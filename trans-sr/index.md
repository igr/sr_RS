# trans-sr

[![npm version](https://badge.fury.io/js/trans-sr.svg)](https://badge.fury.io/js/trans-sr)

See:

	trans-sr --help

Usage:

	trans-sr [options] <input>
		-o, --output <filename>
		-m, --map <conversion map name>
		-f, --file
		-h, --help
		-q, --quiet

## Development

	yarn install
	yarn link
	... work ...
	yarn unlink

## Installation

	yarn install -g

## Publishing

	yarn publish
